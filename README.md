# React ts  开发模板

### `yarn install`
>安装依赖包
### `yarn start`
>运行项目
### `yarn build`
>打包项目
### `yarn eslint`
>检查src下的ts文件错误


### 目录结构
```
src
├── api               接口请求相关
├── components        页面组件
├── config            项目配置相关
├── hooks             hooks
├── index.tsx         入口文件
├── pages             页面
├── redux             redux-saga相关
├── routes            路由配置
├── style             样式
├── ts_study          我的ts学习记录
├── typings           ts定义文件
└── util              工具包
```


### 主要依赖库
1. react react-dom  react-router-dom
2. antd
3. redux redux-saga redux-persist
4. moment
5. Axios
